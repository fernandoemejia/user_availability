# Registration of Username and Restricted Word
 Example with Spring MVC 4, Spring Data JPA, XML Configuration, Maven, JSP, and MySQL.

## Prerequisites
- JDK 1.8 or later
- Maven 3 or later
- MySQL 5.6 or later

## Stack
- Spring MVC

- Spring Data JPA
- Maven
- JSP
- MySQL

## Run
```mvn jetty:run```
